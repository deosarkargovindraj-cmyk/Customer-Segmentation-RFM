Customer Segmentation using RFM Analysis

Description

This project focuses on customer segmentation using RFM (Recency, Frequency, Monetary) analysis. It helps identify valuable customers based on their purchasing behavior.

Objectives

- To analyze customer transaction data
- To calculate RFM metrics
- To group customers using K-Means clustering
- To visualize different customer segments

 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

 Dataset

The dataset includes:

- Customer ID
- Transaction Date
- Purchase Amount

 Methodology

1. Data Cleaning and Preprocessing
2. Calculation of Recency, Frequency, Monetary values
3. Feature Scaling
4. Applying K-Means Clustering
5. Data Visualization

 Results

- Customers are divided into segments like:
  - High-value customers
  - Medium-value customers
  - Low-value
